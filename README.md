# The Shoppies

This is a web application that allow user to search for movies and user can nominate 5 movies. The result will be save in localStorage.

## Technology Used :
* HTML 5
* CSS
* javaScript
* Bootstrap 4
* node.js
* express.js
* OMDb API
* Google Font
* Icons | Font Awesome 
* Icon 8


[Live link](https://shoppies-movies-app.herokuapp.com)
